Ten projekt demonstruje integrację Apache Kafka i Apache Flink do przetwarzania danych strumieniowych.

## Wymagania

- Docker i Docker Compose
- Python 3.7+
- Biblioteki Python: kafka-python, matplotlib, pyflink
